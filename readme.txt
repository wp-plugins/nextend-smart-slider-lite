== Installation ==

1. Upload the files to nextend-smart-slider-lite folder into the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new Smart Slider from the new menu in the control panel sidebar
4. Insert a Smart Slider in your post or page by clicking on the Embed a Smart Slider button in the rich text editor or the button in the sidebar in the post/page view.
